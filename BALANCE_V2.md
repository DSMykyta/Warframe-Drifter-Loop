### Негативні

| Джерело | Хімія | Примітка |
|---------|-------|----------|
| Образливий подарунок | **-3 .. -5** | per OFFENSIVE_GIFTS table |
| Порушена обіцянка | **-5** | per promise |
| Відмова романсу | **-5** | одноразово |
| Розрив стосунків | **-30** | `break_up()` |
| Флірт спіймано | **-5** | партнер бачив |

### Daily Cap

**+15 максимум per NPC per day.** Штрафи (від'ємні) не обмежуються.

Реалізація: `add_chemistry()` у `vars.rpy`, трекер `chemistry_gained_today`.

### Mission Chemistry Gate

Місійна хімія (+6) працює **тільки якщо** `days_since_interaction[NPC] <= 3`.
Тобто потрібно хоча б раз поговорити/подарувати/допомогти за останні 3 дні.

Це запобігає "mission-only" стратегії.

## 3. DECAY (РОЗПАД СТОСУНКІВ)

| Днів без взаємодії | Штраф/день/NPC |
|--------------------|----------------|
| 0-6 | 0 |
| 7-9 | **-1** |
| 10-13 | **-2** |
| 14+ | **-3** |

- **Grace period:** `reset_interaction(name)` скидає лічильник до 0
- Будь-яка дія (розмова, подарунок, місія, допомога) тригерить `reset_interaction()`
- `decay_paused_until` — пауза після key events (rank up, milestones)

### Mission Neglect (відсутність місій)

| Днів без місії | Штраф |
|----------------|-------|
| 0-4 | 0 |
| 5 | **-2 всім** NPC (одноразово) |
| 6+ | **-3 всім** NPC (одноразово, **ресет лічильника**) |

Ресет лічильника означає: після штрафу на день 6, лічильник скидається до 0, новий цикл починається. Це запобігає нескінченному -5/день.

## 7. ПРОГНОЗОВАНІ СЦЕНАРІЇ

### Математична модель

Для кожного NPC, середній щоденний приріст хімії:
- **Місія:** +6 × (частота партнерства) × (mission gate pass rate)
- **Подарунок:** +7 × (частота дарування)
- **Діалог:** +2 × (частота розмов)
- **Decay:** -[1..3] × (пропущені дні)

### Сценарій A: Хороший гравець (взаємодія кожні 2 дні)

| Джерело | /день/NPC |
|---------|-----------|
| Місія (кожні 2 дні) | +3.0 |
| Подарунок liked (кожні 2 дні) | +3.5 |
| Діалог (кожні 2 дні) | +1.0 |
| Decay | ~-0.2 (мінімальний) |
| **Net** | **~+7.3** |

- **День 120 хімії: ~17**
- **Кінцева хімія: ~200+**
- Комфортна перемога, простір для романсу

### Сценарій B: Середній гравець (кожні 3 дні)

| Джерело | /день/NPC |
|---------|-----------|
| Місія (кожні 2.5 дні) | +2.4 |
| Подарунок liked (кожні 3 дні) | +2.3 |
| Діалог (кожні 3 дні) | +0.7 |
| Decay (після дня 7) | ~-0.5 |
| **Net** | **~+4.9 (дні 1-7), ~+4.4 (дні 8+)** |

- Дні 1-7: 4.9 × 7 = **34.3**
- Дні 8-30: 4.4 × 23 = **101.2**
- **Кінцева хімія: ~135**
- **Перемога на день ~27**

### Сценарій C: Недбалий гравець (кожні 4 дні)

| Джерело | /день/NPC |
|---------|-----------|
| Місія | +1.5 |
| Подарунок | +1.75 |
| Діалог | +0.5 |
| Decay | ~-0.75 |
| **Net** | **~+3.0** |

- **Кінцева хімія: ~90**
- **ПРОВАЛ** (120 не досягнуто)
- Потрібна корекція курсу в останні 5-7 днів

### Сценарій D: Тільки місії (без розмов/подарунків)

- +6/день/NPC якщо всі 6 місій (945 хв/960 доступних)
- Але **mission gate** вимагає розмови за 3 дні → без розмов gate не пропускає
- **ПРОВАЛ** — mission chemistry не нараховується

### Сценарій E: Романс (1 NPC до 160 + решта до 120)

- Фокус NPC: mission +6 + loved gift +12 + great dialogue +4 = **+22** (cap: +15/день)
- 160 / 15 = **~11 днів** до романсу (з daily cap)
- Решта 5 NPC: стандартна ротація, ~19 залишкових днів для 120
- **Можливо**, але вимагає планування

## 10. ГАЙД ДЛЯ НАПИСАННЯ НОВОГО КОНТЕНТУ

### Діалоги

```renpy
# Добрий вибір — стандарт
$ add_chemistry("Артур", 2)

# Відмінний вибір — рідко, для емоційних моментів
$ add_chemistry("Артур", 4)

# Поганий вибір
$ add_chemistry("Артур", -2)

# НІКОЛИ не використовуй chemistry["Артур"] += напряму!
# Це обійде daily cap і milestone rep.
```

### Час

```renpy
# Рекомендований час за репліку — 3 хв
$ advance_time(3)

# Для коротких реакцій
$ advance_time(2)

# Для довгих монологів
$ advance_time(5)
```

### Milestones

Milestone діалоги (trust, friends) автоматично доступні через chemistry_min conditions. Rep бонус нараховується автоматично через `add_chemistry()` коли хімія перетинає поріг.

### Подарунки

```python
# inventory.rpy — GIFT_REACTIONS
# Loved (+12): дорогі предмети 300+ крон, по 1-2 на NPC
# Liked (+7): середні предмети 100-250 крон
# Small (+3): дешеві предмети 50-80 крон
# Neutral (+1): все що не в таблицях
```
