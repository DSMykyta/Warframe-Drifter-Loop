## 4. ХІМІЯ: РОЗПАД (DECAY)

| Днів без взаємодії | Втрата/день/NPC |
|--------------------|-----------------|
| 0–6 | 0 |
| 7–9 | **-1** |
| 10–13 | **-2** |
| 14+ | **-3** |

**Grace period:** 2 дні після будь-якої взаємодії (розмова, місія, подарунок) — лічильник decay не рахується.

**Порівняння з аналогами:**
- Stardew Valley: -2/день без розмови (але безкінечний час)
- Persona 5: Нуль decay
- Persona 3 Reload: Reverse після 60 днів neglect
- **Наша гра:** -1 після 7 днів — м'якше за SDV, жорсткіше за Persona. Але при 30-денному ліміті — достатньо відчутно.
